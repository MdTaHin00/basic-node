# Lodash Project 
Lodash Project is a utility library for JavaScript, inspired by Lodash, that provides helpful functions for common programming tasks. This package is designed to simplify your code and improve productivity.

## Installation

To install the package, use npm:

```bash
npm install 22-lodash-project
```

## Usage

Here is an example of how to use the package:

```javascript
const {chunk} = require('lodash-project')

// Example: Using a utility function
const numbers = [1, 2, 3, 4,5];

const size = 2;
console.log(numbers,size); // Output: [[1,2],[3,4],[5]]
```
